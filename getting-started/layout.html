Getting started with Simpletools\Page\Layout

Simpletools\Page\Layout has been designed to provide you with simple and highly efficient layout engine which works great with Simpletools\Mvc.

Simpletools\Page\Layout doesn't provide any new templating language, instead we have decided to stick to what you already know and will guarantee the best performance - PHP

You can simply code in language you already know and be sure performance won't be impacted by any extra layer.

Requirements

Simpletools framework requires PHP version >= 5.3.0

Setup

Simpletools\Page\Layout can be used as a standalone class however for purpose of that document we assume that you are using Simpletools\Mvc alongside hence will provide examples based on that.
